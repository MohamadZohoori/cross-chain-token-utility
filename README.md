# Cross-Chain Token Utility Platform

Production-ready monorepo showcasing end-to-end blockchain development
across Solana and EVM ecosystems.

## Tech Stack
- Solana: Rust, Anchor, SPL Tokens
- EVM: Solidity, Hardhat, Ethers.js
- Backend: Node.js, TypeScript
- Frontend: Next.js, React
- Tooling: Git, CI/CD (planned)

## Structure
- contracts-solana: Solana staking program
- contracts-evm: ERC20 + staking contracts
- backend: event indexer and API
- frontend: wallet-connected dApp UI
